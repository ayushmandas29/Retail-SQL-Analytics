# Retail SQL Analytics

This project analyzes retail sales using SQL. It includes a realistic retail
database schema, sample transactional data, and analytical queries to generate
insights such as revenue, customer spending, city-wise performance, and product
sales.

## Files
- schema.sql
- sample_data.sql
- analysis_queries.sql

## How to Use
1. Run schema.sql to create the tables.
2. Run sample_data.sql to insert sample data.
3. Run analysis_queries.sql to get business insights from the data.
